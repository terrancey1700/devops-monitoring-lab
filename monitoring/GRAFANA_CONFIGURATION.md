# Grafana Configuration

## Objective

Visualize Prometheus metrics in dashboards for real-time observability.

---

## What Was Done

- Connected Grafana to Prometheus as a data source
- Imported production-grade dashboards
- Validated dashboards show real-time container/system activity

---

## What Was Validated

Confirmed visibility into:

- CPU utilization
- Memory usage
- Network traffic

Observed metrics during workload activity to confirm real-time updates.

---

## Outcome

Grafana provided end-to-end visibility into the lab stack using Prometheus as the data source.
